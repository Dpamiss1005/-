
## 运行说明（vscode）
 1. 运行 `npm i` 安装所有包
 2. 在api_server文件下：`node app.js` 运行服务器
 3. 在大事件项目文件下：Ctrl+Shift+P 选择 "Express:Host Current Workspace and open in Browser"
 4. 直接登录（or 注册新账户）：  
  用户名：admin1 密码：123456
